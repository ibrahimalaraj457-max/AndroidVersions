# Android Versions App

A clean and responsive Android application that showcases historical Android version releases using a modern `RecyclerView` architecture. This project was developed as part of **Mobile App Development 1 (Assignment 2)**.

## 🚀 Features Implemented

* **Efficient RecyclerView:** Dynamically handles and displays the list of Android versions vertically using `LinearLayoutManager`.
* **Custom CardView Layout:** Designed an elegant and flexible item UI utilizing `CardView` and `ConstraintLayout` to guarantee full responsiveness across different screen sizes.
* **Data Model & Custom Adapter:** Built a structured `AndroidVersion` model and a custom `VersionAdapter` to bind data seamlessly.
* **Alternating Row Colors (Advanced Task):** Enhanced visual readability by applying alternating subtle background colors (`#F5F5F5` and `#FFFFFF`) dynamically across the items.
* **Interactive Item Clicks:** Implemented click listeners on list items that trigger a descriptive `Toast` message displaying the selected Android code name and version number.
